# QA API Testing Project

## 📌 Descripción
Proyecto de testing de una API pública simulando un entorno real de trabajo QA.

## 🧰 Herramientas
- Postman
- GitHub

## 🔍 Alcance de pruebas
Se realizaron pruebas sobre los siguientes endpoints:

- GET /posts
- GET /posts/{id}
- POST /posts

## ✅ Validaciones
- Verificación de status codes
- Validación de estructura JSON
- Análisis de datos de respuesta

## 🐞 Bugs detectados
- Respuestas sin manejo claro de errores
- Falta de validación en creación de datos

## 📂 Documentación
- Casos de prueba documentados
- Reporte de bugs
- Colección de Postman incluida

## 🚀 Objetivo
Simular una experiencia real de testing de APIs aplicando buenas prácticas QA.
